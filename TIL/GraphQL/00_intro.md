# GraphQL

GraphQL은 서버 API를 통해 정보를 주고받기 위해 사용하는 질의 언어(query language)이다. 
GraphQL API는 보통 하나의 엔드포인트를 사용하며, 요청 시 사용하는 질의문에 따라 응답의 구조가 달라진다.
Apollo GraphQL : gql을 구체적으로 사용할 수 있게 도와주는 라이브러리
